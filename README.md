# SIEM Log Analyzer
A SIEM-like log analysis tool with python featuring real-time detection logic, GUI dashboard, and incident reporting capabilities
## Description
A Python-based log analysis tool with GUI dashboard that detects brute-force attacks and suspicious login behavior.
## Features
- Time-based attack detection
- GUI dashboard (Tkinter)
- Severity levels (INFO / WARNING / ALERT)
- Exportable reports
## Tools Used
- Python
- Tkinter
## Use Case
Simulates real-world SOC monitoring and incident detection.
